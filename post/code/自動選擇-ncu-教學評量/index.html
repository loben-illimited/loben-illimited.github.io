自動選擇 NCU 教學評量

前言

因為 仆街 lms 需要做完整個教學評量才可以 初選

所以才寫以下的code去自動選擇選項。

Code

自動選取選項（隨機地）不會自動submit

questionNumber = 14; //問題總數
for (var i = 1; i <= questionNumber; i++){
    opitionNumber = document.getElementsByName("editAnswer["+ i.toString() +"][0]").length
    selectionAnswer = parseInt(Math.random() * (opitionNumber) ); //隨機選擇選項
    document.getElementsByName("editAnswer["+ i.toString() +"][0]")[selectionAnswer].click()
}

自動選取選項（隨機地）並且會自動submit

questionNumber = 14; //問題總數
for (var i = 1; i <= questionNumber; i++){
    document.getElementsByName("editAnswer["+ i.toString() +"][0]")[0].click()
}
document.getElementsByName("submit")[0].click()

自動選取所有問題第一個選項

即是選取 無缺課 、非常同意等 非常積極的用語。

questionNumber = 14; //問題總數
for (var i = 1; i <= questionNumber; i++){
    document.getElementsByName("editAnswer["+ i.toString() +"][0]")[0].click()
}

使用教學

  1. 在文章中複製需要的code

  1. 然後開啟 仆街 教學評量頁面，並按下 F12 ，打開主控台

  1. 將複製的code貼上在主控台，再按下 enter 鍵

  1. 執行完貼上的code，然後就能看到結果了
